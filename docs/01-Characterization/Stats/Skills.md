# Skills
________________________________________
________________________________________

## Main Info
Characters can obtain 12 of 22 total skills

Note: 
-   Roll Percentages and Roll Edits (RE) are used here
-   Soc: short for “Social Skill”
### Basics
Points:
-   Max points at 30pts
-   Skill specific abilities unlock at 15(M) and 30(M) points

When selecting or adding skills:
-   If a Class, Demi, Genetic Inclusion, or Species you gain has a skill, is automatically added if space is available.
-   If there are 2 of the same skill from different sources, the extra occurrence is added as an extra point for that skill.
	-   This continues for however many instances there are.

### Uses
Skill# = # of points to use in certain activities or actions
When using skills in rolls:
-   # cannot be fully used more than once.
-   # can be divided between ACC and PWR.
-   Number can be split between multi-hits.
You CANNOT use more than 1 skill for the same roll.
